## Running Python Code in Google Colab

You can easily run this Python code using **Google Colab**.

### Step 1: Open Google Colab

Go to: https://colab.research.google.com
Sign in with your Google account and create a **New Notebook**.

### Step 2: Add a Code Cell

Paste your Python code into the code cell.

### Step 3: Run the Code

Click the **Run** button on the left side of the cell
or press **Shift + Enter**.

The output will appear below the code cell.

### Step 4: Save Your Notebook

Your notebook will automatically be saved in **Google Drive**.
